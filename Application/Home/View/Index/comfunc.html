1、字符串
    
    strpos(string,find) 	函数查找字符串在另一字符串中第一次出现的位置(区分大小写,返回字符串在另一字符串中第一次出现的位置，如果没有找到字符串则返回 FALSE)
    	stripos()  			查找字符串在另一字符串中第一次出现的位置（不区分大小写）
	strripos() 			查找字符串在另一字符串中最后一次出现的位置（不区分大小写）
	strrpos() 			查找字符串在另一字符串中最后一次出现的位置（区分大小写）

    str_replace(find,replace,string,count)  以其他字符替换字符串中的一些字符（区分大小写, count 可选-------对替换数进行计数的变量）
        str_ireplace()         函数执行不区分大小写的搜索

	str_repeat() 函数把字符串重复指定的次数。
		str_repeat("Shanghai",5);	//把字符串 "Shanghai " 重复 5 次

    

    2、数组
    
    array_keys(array,[value,strict])    返回包含数组中所有键名的一个新数组
    array_map(myfunction,array1)        将用户自定义函数作用到数组中的每个值上，并返回用户自定义函数作用后的带有新值的数组
    array_walk(array,myfunction)        对数组中的每个元素应用用户自定义函数
        array_walk 可以没有返回值；array_map要有返回值
    

    3、加密
    
    URL编码加密：urldecode可以正确解密rawurlencode加密的空格; rawurldecode不能正确解密urlencode加密的空格
    urlencode(string $str);         //返回字符串,把空格编码为 +
        urldecode(string $str);     //$str:要解码的URL,返回字符串
    rawurlencode(string $str);      //返回字符串,把空格编码为 %20
        rawurldecode(string $str);
    

    4、两位小数格式化
    
    number_format(3.0145,2);//3.01
    sprintf("%.2f", 3.0145); //3.01

    js前端判断(两位小数)
    var num = $("#txt_num").val();
    var dot = num.indexOf(".");
    var dotCnt = num.substring(dot+1,num.length);
    if(!isNaN(num)){
        if(dot != -1){
            if(dotCnt.length > 2){
                alert("小数位已超过2位！");
            }else{
                alert('正确');
            }
        }
    }else{

    }
